CoLoBoT - Niveau supplémentaire #01

- Nécessite une version commerciale 1.6 (ou supérieure).
- Ne fonctionne pas avec une version DEMO.

Normalement, CoLoBoT est installé dans le dossier :

C:\Program Files\Colobot\

Dans ce cas, il faut dézipper l'ensemble des fichiers
dans le dossier :

C:\Program Files\Colobot\user\sample01\
